# Добавление переключения графика температуры

## Что добавлено:

### 1. ✅ Кнопка Restart уже существует
Иконка restart (круговые стрелки) была добавлена ранее и работает корректно.

**Расположение:** `MinerCard.jsx` - рядом с кнопкой "Apply Mode"

### 2. ✅ Переключатель графиков Temperature/Hash Rate

Добавлены две кнопки для переключения между графиками:

**Dashboard.jsx:**
```jsx
<button onClick={() => setChartMetric('hashrate')}>
  📊 Hash Rate
</button>
<button onClick={() => setChartMetric('temperature')}>
  🌡️ Temperature
</button>
```

**Функции:**
- Переключение между хэшрейтом и температурой
- Визуальная индикация активной метрики (синяя кнопка)
- Анимация переходов

### 3. ✅ Универсальный график метрик

**HashRateChart.jsx** теперь поддерживает:

#### Параметры:
- `metricType`: `'hashrate'` или `'temperature'`

#### Конфигурация метрик:
```javascript
const metricConfig = {
  hashrate: {
    title: 'Hash rate',
    unit: 'TH/s',
    color: '#5865f2'
  },
  temperature: {
    title: 'Temperature',
    unit: '°C',
    color: '#f7931a'
  }
};
```

#### Функциональность:
- Автоматическая загрузка данных при переключении метрики
- Динамическое изменение заголовка и единиц измерения
- Поддержка нескольких устройств (цветные линии)
- Кликабельная легенда для скрытия/показа устройств

## API endpoints используемые:

### Temperature:
- `GET /api/metrics/temperature/device/{deviceId}`
- Параметры: `from`, `to`, `bucket`

### Hash Rate:
- `GET /api/metrics/hashrate/device/{deviceId}`
- Параметры: `from`, `to`, `bucket`

## Как использовать:

1. **Переключение метрики:**
   - Нажмите кнопку "📊 Hash Rate" для отображения хэшрейта
   - Нажмите кнопку "🌡️ Temperature" для отображения температуры

2. **Работа с графиком:**
   - Выберите временной диапазон: Day, Week, Month, Year
   - Наведите курсор на график для просмотра точных значений
   - Кликните на устройство в легенде для скрытия/показа его линии

3. **Restart майнера:**
   - Откройте карточку устройства (клик на фиолетовую кнопку)
   - Нажмите иконку с круговыми стрелками справа

## Стили кнопок переключения:

```css
Активная кнопка:
- background: #5865f2
- fontWeight: bold

Неактивная кнопка:
- background: #2a2a2a
- fontWeight: normal
```

## Парсинг данных:

График теперь поддерживает разные форматы данных от бэкенда:
- `point.avgValue` или `point.value` - значение метрики
- `point.timestamp` или `point.time` - временная метка
- Автоматическая конвертация GH/s → TH/s для хэшрейта
- Температура отображается в °C как есть

## Цвета устройств на графике:

```javascript
const COLORS = [

  '#FF5722',  // Красный
  '#9C27B0',  // Фиолетовый
  '#00BCD4',  // Голубой
  '#FFEB3B'   // Желтый
];
```

## Особенности:

1. **Плавное переключение** - график перезагружается при смене метрики
2. **Состояние сохраняется** - выбранные устройства остаются активными
3. **Адаптивность** - график подстраивается под размер экрана
4. **Tooltip** - показывает значения всех устройств в точке

## Известные ограничения:

1. Данные загружаются для всех устройств одновременно (может быть медленно при большом количестве)
2. Максимум 7 уникальных цветов для устройств (дальше повторяются)
3. Температура должна быть в диапазоне 0-150°C для корректного отображения
