## Как использовать

### Вариант 1: Из карточки майнера
1. Откройте Dashboard
2. Разверните карточку нужного устройства
3. Нажмите кнопку **⚡** (справа от кнопки перезагрузки)
4. Выберите режим мощности
5. Укажите IP ASIC (по умолчанию 192.168.3.20)
6. Нажмите **"⚡ Применить режим мощности"**

### Вариант 2: Из детальной страницы
1. Откройте Dashboard
2. Выберите устройство
3. Перейдите на вкладку **"⚡ ASIC Control"**
4. Настройте и примените режим мощности

## Как работает команда

При нажатии кнопки создается команда из 3 шагов:

1. **unlock** - Авторизация на ASIC, получение токена
2. **set_profile_XXXX** - Установка профиля мощности
3. **restart_mining** - Перезапуск майнинга для применения

Токен из шага 1 автоматически подставляется в шаги 2-3.

## История команд

В компоненте есть раздел "📜 История команд" где можно:
- Просмотреть все команды устройства
- Отменить команды со статусом PENDING или SENT
- Повторить команды со статусом FAILED или CANCELLED
- Отслеживать статусы выполнения

### Статусы команд:
- ⏳ **PENDING** - Команда создана, ожидает отправки
- 📤 **SENT** - Команда отправлена на устройство
- ⚙️ **EXECUTING** - Команда выполняется
- ✅ **SUCCESS** - Команда выполнена успешно
- ❌ **FAILED** - Команда завершилась с ошибкой
- 🚫 **CANCELLED** - Команда отменена
